## 作用
当后台检测到***YuanShen.exe***时开始播放音频,播放完音频或播放过程中游戏进程消失(游戏关闭)时,__调用Shell指令删除掉原神资源文件夹__***(没错全删了)***,删除过程相当迅速(很快文件就被删光了)

1.占用较低.(***性能越高的电脑存在感越低!***)
